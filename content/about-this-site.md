---

This site is written and produced by Blake Merryman.

This site is built with [Hugo](https://gohugo.io), a fast open-source static site generator. Pages are written in Markdown and the whole thing is a collection of plain files in a git repository.

The theme is a custom design built from scratch, set in [Geist](https://vercel.com/font) by Vercel. It supports light and dark mode automatically based on your system preference.

The site is hosted on [Vercel](https://vercel.com) and deployed automatically on every push.
